색상 형식 설명

HEX (16진수):

#RRGGBB 형식으로, 웹 디자인에서 가장 많이 사용됩니다.

예: #FF0000 (빨강), #00FF00 (초록), #0000FF (파랑)

RGB (Red, Green, Blue):

각 색상을 0~255 값으로 표현합니다.

예: rgb(255, 0, 0) - 빨강

HSL (Hue, Saturation, Lightness):

  • Hue (색조): 0~360도의 색상환
  • Saturation (채도): 0~100%의 색상 순수도
  • Lightness (명도): 0~100%의 밝기

CMYK (인쇄용):

Cyan, Magenta, Yellow, Key(Black)로 구성된 인쇄용 색상 모드입니다.

색상 조화 이론:

  • 보색: 색상환에서 정반대 위치의 색상
  • 유사색: 색상환에서 인접한 색상들
  • 삼각 조화: 색상환에서 120도씩 떨어진 3개 색상
