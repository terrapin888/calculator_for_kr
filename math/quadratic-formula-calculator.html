이차방정식과 근의공식

이차방정식은 ax² + bx + c = 0 (a ≠ 0) 형태의 방정식입니다.

근의공식

  • 근의공식: x = (-b ± √(b²-4ac)) / 2a
  • 판별식: D = b² - 4ac

판별식에 따른 근의 종류

  • D > 0: 서로 다른 두 실근
  • D = 0: 중근 (하나의 실근)
  • D < 0: 두 허근 (복소수근)

실생활 활용

  • 물리학 (포물선 운동, 최적화 문제)
  • 경제학 (수익 최대화, 비용 최소화)
  • 공학 (구조 설계, 신호 처리)
  • 건축 (아치, 포물선 구조물)
