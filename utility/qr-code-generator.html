QR코드 사용 가이드

📐 크기 선택 가이드

  • 128px: 웹사이트 아이콘, 소셜미디어
  • 256px: 명함, 문서, 일반적 용도
  • 512px: 포스터, 현수막, 대형 인쇄물
  • 1024px: 고해상도 인쇄, 확대 사용

🎨 디자인 팁

  • 전경색과 배경색의 대비를 충분히 확보
  • 어두운 전경, 밝은 배경 권장
  • 컬러 프린터가 없다면 흑백으로 테스트
  • 로고나 이미지 삽입 시 가독성 확인

📱 스캔 최적화

  • 충분한 여백(마진) 확보
  • 최소 2cm × 2cm 이상 크기로 인쇄
  • 평평한 표면에 부착
  • 조명이 충분한 곳에 배치

🔒 보안 고려사항

  • 개인정보가 포함된 QR코드 주의
  • 공개 장소의 QR코드는 신중히 스캔
  • 중요한 정보는 암호화 고려
  • 정기적으로 QR코드 내용 점검

QR코드 종류별 활용법

비즈니스 활용

  • 메뉴판: 레스토랑 디지털 메뉴, 온라인 주문
  • 결제: 간편결제, 계좌이체 정보
  • 리뷰: 구글 리뷰, 만족도 조사 페이지
  • 앱 다운로드: 앱스토어, 구글플레이 링크
  • 소셜미디어: 인스타그램, 페이스북 팔로우

교육 분야

  • 과제 제출: 온라인 과제 제출 페이지
  • 참고자료: 추가 학습 자료 링크
  • 출석체크: 온라인 출석 확인 시스템
  • 설문조사: 수업 만족도, 피드백 수집

이벤트 활용

  • 등록: 행사 사전등록, 참가신청
  • 입장: 전자티켓, 입장권 검증
  • 정보제공: 프로그램 일정, 장소 안내
  • 네트워킹: 참가자 연락처 교환
